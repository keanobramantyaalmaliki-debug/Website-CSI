## Website — MVP 1 (Landing Page)

Tahap pertama pengembangan digital CSI adalah **Landing Page** sebagai MVP 1.

### Tujuan Landing Page
- Memperkenalkan identitas dan positioning CSI
- Menjangkau 5 target audiens utama
- Fondasi sebelum website penuh diluncurkan

### Stack (MVP 1 — Single HTML)
- **Base:** Single HTML file
- **Animation:** GSAP + ScrollTrigger (CDN)
- **3D / WebGL:** Three.js (CDN)
- **Smooth Scroll:** Lenis (CDN)
- **Styling:** Tailwind CSS (CDN)

> Next.js dipakai saat migrasi ke production (post MVP 1)

### Struktur Section (approved)
1. **Loading Screen** — animated entry
2. **Hero** — full viewport, WebGL/3D, tagline "Cogniti"
3. **Manifesto** — visi & belief CSI, bukan deskripsi layanan
4. **Approach** — cara berpikir & bekerja CSI
5. **Who We Serve** — 5 segmen audiens + value prop
6. **CTA / Contact**
7. **Footer**
